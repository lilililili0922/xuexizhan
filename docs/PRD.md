# 学习站 H5 产品需求文档

版本：V1.0.0

## 1. 概述

学习站用于集中承载暑期集训的课程安排、作业入口、阅读分享、每日提醒、早上签到和学生心声。学生端强调快速知道“今天做什么、现在去哪、谁到了、我可以反馈什么”；老师后台强调低成本维护课程和班级到课情况。

## 2. 核心角色

| 角色 | 权限 |
| --- | --- |
| 学生 | 浏览课程、查看签到汇总、提交心声、点赞/举报评论 |
| 老师/助教 | 维护课程、每日提醒、阅读分享人、签到表、评论审核 |
| 超级管理员 | 后续扩展账号和权限管理 |

## 3. 功能清单

| 模块 | 功能 | 优先级 | 说明 |
| --- | --- | --- | --- |
| 今日课程 | 展示当天课程和会议入口 | P0 | 自动区分进入会议、加入会议列表、已结束、看回放 |
| 本周课程 | 展示周课表 | P0 | 按当前周过滤和排序 |
| 快捷入口 | 作业、资源、资料、回放、请假 | P0 | 全部外链新标签打开，禁止 `#` 占位 |
| 每日内容 | 阅读分享人、注意事项、首页标语 | P0 | 老师后台维护 |
| 早上签到表 | 已到/迟到/未到汇总和明细 | P0 | 学生端可看清到课情况，老师后台可更新 |
| 学生心声角 | 匿名评论、点赞、举报 | P1 | 微信身份校验后发布，前台匿名 |
| 评论审核 | 老师处理 pending/hidden/deleted | P1 | 后台保留微信 hash 追溯 |

## 4. 早上签到表需求

### 学生端

- 展示总人数、已到人数、迟到人数、未到/缺勤人数。
- 展示迟到总结：有迟到时列出姓名；无迟到时显示“今天暂未记录迟到同学”。
- 展示明细表：学生、班型、到达时间、状态、备注。
- 状态枚举：已到、迟到、缺勤、未到。

### 老师后台

- 每个学生一行，可填写到达时间和备注。
- 支持一键标记：已到、迟到、缺勤、清除。
- 支持维护当天迟到截止时间，默认 `09:30`。
- 更新后学生端刷新即可看到最新结果。

## 5. 验收标准

| 场景 | 验收条件 |
| --- | --- |
| 首页课程 | 2026-07-02 显示第 1 周、今日 3 节课程、本周 11 节课程 |
| 签到汇总 | 学生端能看到已到、迟到、未到/缺勤数量 |
| 迟到总结 | 有迟到学生时展示迟到学生姓名 |
| 后台签到 | 老师标记某学生迟到后，学生端签到表刷新后同步更新 |
| 心声发布 | 未登录提示微信登录，登录后匿名发布 |
| 评论审核 | 后台可通过、隐藏、删除评论 |
| 移动端 | 375px 宽度下页面不横向挤压，表格允许横向滚动 |
